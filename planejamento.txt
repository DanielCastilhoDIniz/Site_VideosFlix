* 1° passo - criando ambiente virtual - python -m venv dev_django.
   - ativar ambiente virtual - . dev_django\Scripts\activate.ps1 
* 2° escolher diretório para criar o projeto
* 3° começar proejto com o comando - django-admin startproject nome_do_projeto
* 4° criar app - django-admin startapp nome_do_app
* 5° criar superuser - python manege.py migrate 
    - djando-admin mange.py createsuperuser
* 6° integrar app com proejto -   ir até o arquivo settings.py:
       - # Application definition

                INSTALLED_APPS = [
                    'nome_do_app'
                ]

        - # Internationalization
        # https://docs.djangoproject.com/en/4.2/topics/i18n/

            LANGUAGE_CODE = 'pt-bt' <-------------

            TIME_ZONE = 'America/Sao_Paulo' <-------------

            USE_I18N = True

            USE_TZ = True

        
* 7° criar classes em models.py
    - cada elemento da tabela é criado
    - comando - python manage.py makemigrations  
                python manage.py migrate
