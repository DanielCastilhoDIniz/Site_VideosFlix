Passos:

Criar ambiente virtual:

Execute o comando python -m venv dev_django para criar o ambiente virtual.
Ative o ambiente virtual com o comando . dev_django\Scripts\activate.ps1.
Escolher o diretório para criar o projeto.

Iniciar o projeto:

Execute o comando django-admin startproject nome_do_projeto para começar o projeto Django.
Criar um aplicativo:

Execute o comando django-admin startapp nome_do_app para criar um novo aplicativo.
Criar um superusuário:

Execute os comandos python manage.py migrate e django-admin manage.py createsuperuser para migrar o banco de dados e criar um superusuário.
Integrar o aplicativo ao projeto:

Abra o arquivo settings.py do projeto e adicione o nome do aplicativo ao INSTALLED_APPS.
Defina LANGUAGE_CODE como 'pt-br' e TIME_ZONE como 'America/Sao_Paulo'.
Verifique se USE_I18N e USE_TZ estão definidos como True.
Criar classes no arquivo models.py:

Crie uma classe para cada elemento da tabela.
Execute os comandos python manage.py makemigrations e python manage.py migrate para criar e aplicar as migrações.
Configurar arquivos estáticos:

Crie a pasta static no diretório do projeto com as subpastas css, js e images.
No arquivo settings.py, ajuste STATIC_URL para 'static/' e adicione STATICFILES_DIRS com o caminho para a pasta static.
No arquivo urls.py do projeto, adicione + static(settings.STATIC_URL, document_root=settings.STATIC_ROOT) ao urlpatterns e importe as bibliotecas necessárias.
Configurar arquivos de mídia:

Crie a pasta media no diretório do projeto.
No arquivo settings.py, defina MEDIA_URL e MEDIA_ROOT com os caminhos corretos para a pasta media.
No arquivo urls.py do projeto, adicione + static(settings.MEDIA_URL, document_root=settings.MEDIA_ROOT) ao urlpatterns.
Criar a pasta templates na raiz do projeto:

No arquivo settings.py, adicione 'templates' em 'DIRS' dentro das configurações do 'TEMPLATES'.
Essa pasta será usada como o diretório de templates para toda a aplicação.
Criar a pasta templates dentro do aplicativo filme:

Essa pasta terá o padrão para o aplicativo filme.
